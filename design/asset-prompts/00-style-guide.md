# Style commun (déjà inclus dans chaque prompt)

Tous les prompts partagent ce socle de style — inutile de le rajouter, il est
déjà collé dans chaque fichier. Il est ici pour référence si tu veux ajuster.

**Style :**
> 3D rendered mobile game asset, Supercell art style (Clash Royale / Brawl
> Stars), glossy and polished, soft rounded forms, vibrant saturated colors,
> smooth studio lighting with a gentle rim light, subtle glow, cute and playful,
> clean readable silhouette, thick premium bevels.

**Cadrage & fond (crucial pour le détourage) :**
> Single centered subject with ~12% empty padding on all sides. FLAT SOLID
> background, pure magenta #FF00FF, perfectly uniform — no gradient, no texture,
> NO drop shadow or reflection cast on the background. 1024×1024, PNG.

**Pourquoi magenta ?** Aucune de nos images n'est magenta, donc le contraste est
maximal pour un détourage auto propre. Si jamais un sujet contient du rose/violet
vif qui se confond, remplace le fond par un **vert `#00B140`** (indiqué dans les
prompts concernés).
